1. `npm run type-check`로 TypeScript 코드들을 체크하고 문제가 생기는 부분 중, 이유가 명확한 것들은 수정합니다. 재검토가 필요하다고 생각되는 코드들은 정리해서 알려주세요. (type-check 에는 tsc --watch 가 포함되어 있습니다)
2. `npm run lint`로 코드들을 Linting 합니다. 역시 수정 가능한 것들은 수정해주세요.
3. type-check와 lint에서 문제가 없다면 `npm run test`를 실행해서 에러가 없는지 체크해주세요. 역시 수정 가능한 것들은 수정해주세요.
